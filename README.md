# LIS545-TermProject-Scheidegger
This repository includes files relating to SJ Scheidegger’s final LIS 545 Term Project. This Term Project is based on the dataset from the project *Cave Temperature and Air flow data (Longeaigue Cave, Jura Mountains, Switzerland) for Dynamics of chimney-effect ventilation and heat exchange,* by Claudio Pastore, Amir Sedaghatkish, Eric Weber, Marc Luetscher, and Pierre-Yves Jeannin. This dataset is published on Zenodo at https://zenodo.org/records/18166879. 

The files contained within this repository are:

### Data Files (folder)
Original Excel files for the chosen dataset.

### Final Report
Contains the Data and Metadata Profile and Repository Profile for the chosen dataset along with a recommended data citation, long-term preservation considerations, note about copyright, and a statement on human subject considerations related to this dataset. 

### Metadata File
XML file using the DataCite metadata schema to describe the dataset.
